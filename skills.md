# Project Skills

This file contains custom skills and routines specific to the HireIQ Interview platform.

You can instruct Gemini to perform complex, repeated tasks by referencing skills in this file, or you can run custom scripts from here.

## 1. Local Development
- Command to start backend: `cd backend && uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload`
- Command to start frontend: `cd Front-end && npm run dev`

## 2. Deployment
- The backend should be deployed to **Render Web Services** (Free Tier). 
- The frontend should be deployed to **Vercel**.
- Database is hosted on **MongoDB Atlas**.
